Bundled RalphieNode Firmware Artifacts
======================================

These files are placeholders in source control and must be replaced by signed
release firmware during packaging.

Fail-visible behavior:
- If the artifact is missing, onboarding fails explicitly.
- If the artifact is invalid for the detected hardware profile, flashing fails explicitly.

Profiles:
- `ralphienode-esp32-generic.bin`
- `ralphienode-heltec-v3.bin`
- `ralphienode-rp2040.uf2`
